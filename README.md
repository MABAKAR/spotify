# INSTALL

CLONE THIS REPO

`git clone git@github.com:MABAKAR/spotify.git `


Navigate in the folder

`cd spotify`



#### IMPORT THE FILE SQL.SQL WITH PHPMYADMIN

`cd api`

Insatll project dependencies

`composer install`

`symfony serve -d`


Navigate to the folder frontend

`cd ../frontend`

Install project dependencies

`npm install`

launch the frontend

`npm start`
